# 建立家庭會員（子會員）注意事項

> 櫃檯 / 員工速查。子會員 = 家庭成員（`isChildAccount`，以 `parentMemberId` 綁定家長）。
> 規則皆為**後端權威強制**（前端提示為輔）；相關入場資格見 `entry-eligibility-flow.md`。

## 一、建立方式
- **會員自己建**：App →「個人 / 家庭成員」→ 新增（`POST /members/my/children`）。
- **店員建**：會員資料頁 →「家庭成員」新增（`POST /:id/children`）。
- 家庭成員**共用家長電話**；系統以 `parentMemberId` 為唯一親子關係判定（不倚賴 `isChildAccount` 旗標）。

## 二、年齡限制（後端強制擋，`memberService.createMember`）
| 規則 | 錯誤碼 | 說明 |
|---|---|---|
| 未滿 5 歲 | `AGE_UNDER_5` | 不能成為會員（含子會員） |
| 家庭成員須未滿 18 歲 | `AGE_RESTRICTION` | 滿 18 歲不能建成子會員，須改註冊**正式會員** |
- **生日務必填正確**：系統以出生日期判定「兒童（age<13）」等資格（非 `memberType`），填錯會影響入場類型與各項限制。

## 三、入場前置（完成才能入場）
1. **風險安全聲明書 + 墜落測驗同意書**：由**法定代理人（家長）代簽**（子會員頁 forChild 模式）。
2. 兩份簽完 → **安排墜落測驗（選場館）**（子會員墜測頁 / MemberProfilePage 家庭成員皆可）→ 進該館站台待辦 → 現場測驗通過。
3. **持當日有效體驗券者**：未過墜測也可入場，但**仍須先簽墜測同意書**（否則 `fall_test_consent_required`）。

## 四、兒童（未滿 13 歲）消費限制（後端擋）
- ❌ 購買**定期票** → `CHILD_NO_PASS`
- ❌ 購買**優惠折扣券 / 優惠卡** → `CHILD_NOT_ALLOWED`
- ❌ **接受點數轉移 / 定期票轉讓** → `CHILD_NOT_ALLOWED`
- ❌ **兒童入場不套任何折扣**（隊員 9 折、舊折扣卡 8 折皆不適用，一律原價）

## 五、櫃檯操作要注意
- **電話搜尋入場**：輸入共用電話 → 系統先解析到**家長** → 需**從家庭成員清單選出該子女**，才會出現「兒童入場」選項（`/checkin/eligibility` 以出生日期判兒童）。
- **子會員票券唯讀**：家長在「我的票券 / 我的課程 / 我的紀錄」看得到子女項目（標「👦 姓名」），但**移轉 / 使用動作只能綁本人**（子女票券後端擁有權綁登入者本人）。
- **一支電話多帳號**：解析優先「家長帳號（非子帳號、無 parentMemberId）」，避免誤操作到子帳號。
- **課程 / 體驗代報名**：家長可代子女報名（帶 `childMemberId`），系統驗證擁有權；未滿 5 歲不能報課程 / 體驗。

## 六、其他
- **Email 可共用**（親子本就共用；系統允許，不做唯一性檢查）。
- **子會員滿 18 歲** → 需走「升級為正式會員」流程（`promote`，`POST /:id/promote`），不再是家庭成員。
- **自動認領**：以電話+姓名比對——子帳號**不走**墜測 / 隊員 / 90 日票自動認領（避免共用電話認錯人）。
